How to run?

1. To deploy / test

- Go to the root directory containing the docker-compsose file.
  Run:

```bash
docker compose up -d --build
```

Frontend is configured to run at port `5173`.

2. To develop

```bash
npm install
npm run dev
```

---

- Node version: ```20.11.1```
- NPM version: ```10.2.4```
